# BOARetriever

Pequeño rastreador diario del boletín de Aragón (BOA) para encontrar palabras clave usando los datos abiertos que proporciona la Administración. Proyecto para volver a retomar la programación en Java.

## Acciones que realiza por el momento

 - Obtener en el BOA del día de hoy cualquier publicación que tenga la palabra "Informática". La palabra puede cambiar dependiendo lo que ponga en el fichero de properties. No es case sensitive.
